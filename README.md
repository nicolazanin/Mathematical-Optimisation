## Get Started
This project uses **Python 3.13**.
#### Installing Required Packages
<ol>
<li>
Create a virtual environment:

 ```
 python3.13 -m venv venv
 source venv/bin/activate 
 ```
</li>

<li>
Install the required packages:

```
pip install -r requirements.txt
```
</li>

<li>
Run <em>test.py</em>:

```
python test.py
```
</li>
</ol>
